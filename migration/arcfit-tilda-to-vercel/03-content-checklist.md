## Что взять с Tilda как файлы

### Изображения
Проверить `public/images/` в Next.js — чего не хватает:
- [ ] Hero фото (`hero/`)
- [ ] Галерея студии (`gallery/`)
- [ ] Фото залов (`facilities/`)
- [ ] Фото тренеров (`trainers/`)
- [ ] Фото с занятий (programs)
- [ ] Логотип (svg)
- [ ] OG-image (1200x630)

Если на Tilda есть фото которых нет в репо — скачать через Tilda CDN, оптимизировать до <300 КБ desktop / <150 КБ mobile, положить в `public/images/`.

### Шрифты
Бренд-шрифты Unbounded + Inter — должны загружаться через `next/font`. Проверить `src/app/layout.tsx`.

### Метатеги
- [ ] `<title>` каждой страницы есть (`metadata.title`)
- [ ] `<meta description>` (`metadata.description`)
- [ ] OG-теги (`openGraph`)
- [ ] Schema.org разметка (есть на главной — SportsActivityLocation)

## Сравнение «бок-о-бок»

Перед миграцией можно поднять preview на поддомене staging.arcfit.ru или просто использовать `arcfit-next.vercel.app` для сравнения. Открыть в одном браузере две вкладки и пройтись по каждой странице.

```bash
# Скачать главную Tilda и Next.js, diff
curl -s https://arcfit.ru/ > /tmp/tilda.html
curl -s https://arcfit-next.vercel.app/ > /tmp/next.html
# визуальное сравнение в Chrome MCP
```
